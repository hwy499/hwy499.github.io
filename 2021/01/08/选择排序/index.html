选择排序

基本思想

首先在未排序序列中找到最小（大）元素，存放到排序序列的起始位置。
再从剩余未排序元素中继续寻找最小（大）元素，然后放到已排序序列的末尾。
重复第二步，直到所有元素均排序完毕。

代码实现

static void selectSort(int[] src) {
    int len = src.length;
    // 比较 len - 1 词
    for (int i = 0; i < len - 1; i++) {
        // 记录当前下标，作为之后比较使用
        int min = i;
        // 找出最小的元素,需要比较 len - 1
        for (int j = i + 1; j < len; j++) {
            if (src[j] < src[min]) {
                min = j;
            }
        }
        // 如果最小元素下标发生改变，则交换
        if (min != i) {
            int temp = src[i];
            src[i] = src[min];
            src[min] = temp;
        }
    }
}
