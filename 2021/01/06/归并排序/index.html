归并排序

归并排序（Merge Sort）是建立在归并操作上的一种有效，稳定的排序算法，该算法是采用分治法（Divide and Conquer）的一个非常典型的应用。将已有序的子序列合并，得到完全有序的序列；即先使每个子序列有序，再使子序列段间有序。若将两个有序表合并成一个有序表，称为二路归并。

基本思想

1、申请空间，使其大小为两个已经排序序列之和，该空间用来存放合并后的序列；
2、设定两个指针，最初位置分别为两个已经排序序列的起始位置；
3、比较两个指针所指向的元素，选择相对小的元素放入到合并空间，并移动指针到下一位置；
4、重复步骤 3 直到某一指针达到序列尾；
5、将另一序列剩下的所有元素直接复制到合并序列尾。

代码实现

// 排序入口函数
public static void mergeSort(int[] arr) {
    if (arr == null || arr.length < 2) {
        return;
    }
    sortProcess(arr, 0, arr.length - 1);
}
// 分治
static void sortProcess(int[] arr, int left, int right) {
    if (right == left) {
        return;
    }
    int mid = left + ((right - left) >> 1);
    // 归并左边
    sortProcess(arr, left, mid);
    // 归并右边
    sortProcess(arr, mid + 1, right);
    merge(arr, left, mid, right);
}
// 合并
static void merge(int[] arr, int left, int mid, int right) {
    // 辅助数组
    int[] help = new int[right - left + 1];
    // 辅助数组 开始下标
    int i = 0;
    int p1 = left;
    int p2 = mid + 1;
    // 合并两个数组
    while (p1 <= mid && p2 <= right) {
        help[i++] = arr[p1] <= arr[p2] ? arr[p1++] : arr[p2++];
    }
    // 两个指针必有一个未过界 需要处理
    while (p1 <= mid) {
        help[i++] = arr[p1++];
    }
    while (p2 <= right) {
        help[i++] = arr[p2++];
    }
    // 拷贝给原数组
    for (i = 0; i < help.length; i++) {
        arr[left + i] = help[i];
    }
}

复杂度

  • 时间复杂度:O(nlogn)
  • 空间复杂度:O(N) 需要一个与排序数组长度相同的数组，进行辅助排序

稳定性

归并排序是一个稳定的排序，只需在合并方法中，合并两个数组时用这行代码 help[i++] = arr[p1] <= arr[p2] ? arr[p1++] : arr[p2++],可以保证当左右两部分的值相等的时候，先复制左边的值，这样可以保证值相等的时候两个元素的相对位置不变。
