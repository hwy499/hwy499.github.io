leetcode-105-从前序与中序遍历序列构造二叉树

题目描述

根据一棵树的前序遍历与中序遍历构造二叉树。

注意:
你可以假设树中没有重复的元素。

例如，给出

前序遍历 preorder = [3,9,20,15,7]
中序遍历 inorder = [9,3,15,20,7]

返回如下的二叉树：

​ 3

/
9 20
/
15 7

来源：力扣（LeetCode）
链接：https://leetcode-cn.com/problems/construct-binary-tree-from-preorder-and-inorder-traversal
著作权归领扣网络所有。商业转载请联系官方授权，非商业转载请注明出处。

我的想法

假设我们要还原的树是下图

前序遍历： [3, 9, 20, 15, 7 ]

前序遍历：首先遍历根节点，然后是左子树，最后是右子树。

中序遍历 [ 20, 9, 15, 3, 7 ]

中序遍历首先遍历左子树，然后是根节点，最后是右子树。

前序遍历先访问的是根节点，所以节点3就是根节点。由于中序遍历是在左子树都遍历完的时候才遍历根节点，所有在中序遍历中节点3前面的都是节点3的左子树节点，后面的都是节点3的右子树节点。也就是下面这样。

我们再使用同样的方式对左右子树继续划分，一直这样下去，直到不能再分为止，我们来看下代码。

我的代码

方法一：递归方式实现

import java.util.HashMap;

/**
 * 二叉树节点
 */
class TreeNode {
    int val;
    TreeNode left;
    TreeNode right;

    TreeNode() {
    }

    TreeNode(int val) {
        this.val = val;
    }

    TreeNode(int val, TreeNode left, TreeNode right) {
        this.val = val;
        this.left = left;
        this.right = right;
    }
}

public class Solution {

    /**
     * 利用hash表更快定位根节点在中序遍历中的位置
     */
    private HashMap<Integer, Integer> rootIndex;

    /**
     * 功能：根据前序遍历以及中序遍历构建一个完整的二叉树
     *
     * @param preorder
     * @param inorder
     * @return
     */
    public TreeNode buildTree(int[] preorder, int[] inorder) {
        rootIndex = new HashMap<>();
        // 维护每个节点在中序遍历中的索引，便于在递归中寻找根节点root在中序遍历结果中的位置
        for (int i = 0; i < inorder.length; i++) {
            rootIndex.put(inorder[i], i);
        }
        return obtainTree(preorder, inorder, 0, preorder.length - 1, 0, inorder.length - 1);
    }

    /**
     * 功能：根据前序遍历以及中序遍历构建一个完整的二叉树
     *
     * @param preorder
     * @param inorder
     * @param pre_left
     * @param pre_right
     * @param in_left
     * @param in_right
     * @return
     */
    public TreeNode obtainTree(int[] preorder, int[] inorder, int pre_left, int pre_right, int in_left, int in_right) {
        // 出口条件
        if (pre_left > pre_right) {
            return null;
        }
        // pre_left 是每一个子树根节点所在的位置,根据 pre_left 初始化每一个子树的根节点
        TreeNode root = new TreeNode(preorder[pre_left]);
        // 获取该节点在中序遍历中的位置
        int root_index = rootIndex.get(root.val);
        // 获取子树中左子树中节点的个数
        int leftSubtreeNodeNum = root_index - in_left;
        // 递归构建 root 的左子树
        // pre_left +1 -> pre_right + leftSubtreeNodeNum 是以 root 为根节点的左子树的先序遍历结果
        // in_left -> root_index - 1 是以 root 为根节点的子树的中序遍历结果
        root.left = obtainTree(preorder, inorder, pre_left + 1, pre_left + leftSubtreeNodeNum, in_left, root_index - 1);
        // 递归构建root的右子树
        // pre_left + leftSubtreeNodeNum + 1 -> pre_right 是以 root 为根节点的右子树的先序遍历结果
        // root_index + 1 -> in_right 是以 root 为根节点的右子树的中序遍历结果
        root.right = obtainTree(preorder, inorder, pre_left + leftSubtreeNodeNum + 1, pre_right, root_index + 1, in_right);
        // 构建完root的左右子树，将root返回
        return root;
    }

    /**
     * 前序遍历
     *
     * @param node
     */
    public void preOrder(TreeNode node) {
        if (node == null) {
            return;
        }
        System.out.print(node.val + " ");
        preOrder(node.left);
        preOrder(node.right);
    }

    /**
     * 中序遍历
     *
     * @param node
     */
    public void inOrder(TreeNode node) {
        if (node == null) {
            return;
        }
        inOrder(node.left);
        System.out.print(node.val + " ");
        inOrder(node.right);
    }

    public static void main(String[] args) {
        Solution solution = new Solution();
        int preorder[] = new int[]{3, 9, 20, 15, 7};
        int inorder[] = new int[]{20, 9, 15, 3, 7};
        TreeNode root = solution.buildTree(preorder, inorder);
        System.out.println("前序遍历:");
        solution.preOrder(root);
        System.out.println("\n中序遍历:");
        solution.inOrder(root);
    }
}

复杂度：时间复杂度：O(n)，空间复杂度：O(n)

方法二：利用栈实现（递归实现的方法总是能用栈的方式实现）

import java.util.Stack;

/**
 * 二叉树节点
 */
class TreeNode {
    int val;
    TreeNode left;
    TreeNode right;

    TreeNode() {
    }

    TreeNode(int val) {
        this.val = val;
    }

    TreeNode(int val, TreeNode left, TreeNode right) {
        this.val = val;
        this.left = left;
        this.right = right;
    }
}

public class Solution {

    /**
     * 功能：根据前序遍历以及中序遍历构建一个完整的二叉树
     *
     * @param preorder
     * @param inorder
     * @return
     */
    public TreeNode buildTree(int[] preorder, int[] inorder) {
        if (preorder == null || preorder.length == 0) {
            return null;
        }
        Stack<TreeNode> stack = new Stack<>();
        TreeNode root = new TreeNode(preorder[0]);
        stack.push(root);
        int index = 0;
        for (int i = 1; i < preorder.length; i++) {
            TreeNode temp = stack.peek();
            // 如果栈顶元素不等于此时的中序元素 说明要构建右子树了 等于的话就接着构建构建左子树
            if (temp.val != inorder[index]) {
                // 初始化节点的左子树
                temp.left = new TreeNode(preorder[i]);
                // 将此压入栈中
                stack.push(temp.left);
            } else {
                // 寻找右子树的根节点 如果栈顶元素不等于此时的中序元素，说明找到了
                while (!stack.isEmpty() && stack.peek().val == inorder[index]) {
                    temp = stack.pop();
                    index++;
                }
                // 初始化节点的右子树
                temp.right = new TreeNode(preorder[i]);
                // 将此压入栈中
                stack.push(temp.right);
            }

        }
        return root;
    }

    /**
     * 前序遍历
     *
     * @param node
     */
    public void preOrder(TreeNode node) {
        if (node == null) {
            return;
        }
        System.out.print(node.val + " ");
        preOrder(node.left);
        preOrder(node.right);
    }

    /**
     * 中序遍历
     *
     * @param node
     */
    public void inOrder(TreeNode node) {
        if (node == null) {
            return;
        }
        preOrder(node.left);
        System.out.print(node.val + " ");
        preOrder(node.right);
    }

    public static void main(String[] args) {
        Solution solution = new Solution();
        int preorder[] = new int[]{3, 9, 20, 15, 7};
        int inorder[] = new int[]{9, 3, 15, 20, 7};
        TreeNode root = solution.buildTree(preorder, inorder);
        System.out.println("前序遍历:");
        solution.preOrder(root);
        System.out.println("\n中序遍历:");
        solution.inOrder(root);

    }

}

复杂度：时间复杂度：O(n)，空间复杂度：O(n)
