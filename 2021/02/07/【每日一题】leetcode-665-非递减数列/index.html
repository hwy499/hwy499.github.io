leetcode-665-非递减数列

题目描述

给你一个长度为 n 的整数数组，请你判断在最多改变1个元素的情况下，该数组能否变成一个非递减数列。

我们是这样定义一个非递减数列的： 对于数组中所有的 i (0 <= i <= n-2)，总满足 nums[i] <= nums[i + 1]。

示例 1:

输入: nums = [4,2,3]
输出: true
解释: 你可以通过把第一个4变成1来使得它成为一个非递减数列。

示例 2:

输入: nums = [4,2,1]
输出: false
解释: 你不能在只改变一个元素的情况下将其变为非递减数列。

说明：

  • 1 <= n <= 10 ^ 4

  • 10 ^ 5 <= nums[i] <= 10 ^ 5

我的想法

题目要求只能改变一个元素，让数组满足非递减序列。我就只需要遍历数组，当遍历到前一个元素大于后一个元素时，即arr[i]>arr[i+1]时，我就将arr[i+1]的值赋给arr[i]，然后判断数组是否满足非递减序列，满足则返回true，不满足的话将arr[i]的值恢复原状，将arr[i]的值赋值给arr[1+1]，然后在判断此时的数组是否满足非递减序列，然后将此结果返回。

代码实现

class Solution {

    public boolean checkPossibility(int[] nums) {
        // 如果只有一个数字就直接返回true
        if (nums.length == 1) {
            return true;
        }
        // 遍历整个数组
        for (int i = 0; i < nums.length - 1; i++) {
            int curr = nums[i];
            int next = nums[i + 1];
            // 如果前一个大于后一个，不满足非递减
            if (curr > next) {
                // 将前一个的值改为后一个的值
                nums[i] = next;
                // 此时对数组进行判断
                if (checkArray(nums)) {
                    return true;
                }
                // 还原前一个值
                nums[i] = curr;
                // 将后一个的值改为前一个
                nums[i + 1] = curr;
                // 然后对数组进行判断，并将判断结果作为程序的结果返回
                return checkArray(nums);
            }
        }
        // 如果没有发生改变，说明本身就是非递减序列
        return true;
    }

    /**
     * 判断一个数组是不是非递减数组
     * @param arr
     * @return
     */
    private boolean checkArray(int[] arr) {
        for (int i = 0; i < arr.length - 1; i++) {
            if (arr[i] > arr[i + 1]) {
                return false;
            }
        }
        return true;
    }
}
