leetcode-547-省份数量

题目描述

有 n 个城市，其中一些彼此相连，另一些没有相连。如果城市 a 与城市 b 直接相连，且城市 b 与城市 c 直接相连，那么城市 a 与城市 c 间接相连。

省份 是一组直接或间接相连的城市，组内不含其他没有相连的城市。

给你一个 n x n 的矩阵 isConnected ，其中 isConnected[i][j] = 1 表示第 i 个城市和第 j 个城市直接相连，而 isConnected[i][j] = 0 表示二者不直接相连。

返回矩阵中 省份 的数量。

示例 1：

输入：isConnected = [[1,1,0],[1,1,0],[0,0,1]]
输出：2

示例 2：

输入：isConnected = [[1,0,0],[0,1,0],[0,0,1]]
输出：3

提示：

  • 1 <= n <= 200
  • n == isConnected.length
  • n == isConnected[i].length
  • isConnected[i][j] 为 1 或 0
  • isConnected[i][i] == 1
  • isConnected[i][j] == isConnected[j][i]

解题思路

我们可以将这个题转换为求无向图中的连通域的个数， isConnected 二维数组即为该无向图的邻接矩阵。求无向图中连通域的个数的求法就有好多种了，常见的有深度优先遍历(DFS)、广度优先遍历(BFS)、并查集

解法

深度优先遍历：

class Solution {
    public int findCircleNum(int[][] isConnected) {
        // 城市的总个数
        int total = isConnected.length;
        // 根据城市总个数初始化一个访问数组
        boolean[] visited = new boolean[total];
        // 统计省份个数的
        int result = 0;
        // 遍历每一个城市
        for (int i = 0; i < total; i++) {
            // 如果当前城市i未被访问过，则遍历与城市i相邻的所有城市
            if (!visited[i]) {
                // 遍历和城市i相连接的城市
                dfs(isConnected, visited, total, i);
                // 结果加一
                result++;
            }
        }
        return result;
    }

    /**
     * 功能：寻找与城市i所有相连的城市，并在visited数组中标记
     * @param isConnected 所有城市组成的无向邻接矩阵
     * @param visited 访问数组 标记当前城市是否被访问过
     * @param total 城市的总个数
     * @param i 当前要遍历的城市i
     */
    private void dfs(int[][] isConnected, boolean[] visited, int total, int i) {
        for (int j = 0; j < total; j++) {
            // 如果两个城市相连接，并且该城市未访问过，则将j城市标记为访问过，在接着找和j城市相连接的城市
            if (isConnected[i][j] == 1 && !visited[j]) {
                // 将城市j标记为访问过
                visited[j] = true;
                // 继续找和j城市相连的城市
                dfs(isConnected, visited, total, j);
            }
        }
    }
}

广度优先遍历

class Solution {
    public int findCircleNum(int[][] isConnected) {
        // 城市的总个数
        int total = isConnected.length;
        // 根据城市总个数初始化一个访问数组
        boolean[] visited = new boolean[total];
        // 统计省份个数的
        int result = 0;
        // 定义一个队列
        Queue<Integer> queue = new LinkedList<>();
        for (int i = 0; i < total; i++) {
            // 如果城市i没有被访问过，则要访问它的所有相连的城市
            if (!visited[i]) {
                // 将城市i加入到队列中，等待访问
                queue.offer(i);
                // 将城市i的访问状态设置成true
                visited[i] = true;
                // 如果队列不空，就要访问队列中所有城市的相连城市
                while (!queue.isEmpty()) {
                    // 取出城市v
                    int v = queue.poll();
                    // 遍历与城市v相连的所有城市
                    for (int j = 0; j < total; j++) {
                        // 如果j城市与v城市相连，且j城市未被访问过，就将城市j的访问状态置为true，
                        // 并将j城市放入队列中，等待访问
                        if (isConnected[v][j] == 1 && !visited[j]) {
                            visited[j] = true;
                            queue.offer(j);
                        }
                    }
                }
                // 访问玩城市i相连的城市了，省份加一
                result++;
            }
        }
        return result;
    }
}
