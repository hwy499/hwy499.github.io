leetcode-84-柱状图中最大的矩形

题目描述

给定 n 个非负整数，用来表示柱状图中各个柱子的高度。每个柱子彼此相邻，且宽度为 1 。

求在该柱状图中，能够勾勒出来的矩形的最大面积。

以上是柱状图的示例，其中每个柱子的宽度为 1，给定的高度为 [2,1,5,6,2,3]。

图中阴影部分为所能勾勒出的最大矩形面积，其面积为 10 个单位。

示例:

输入: [2,1,5,6,2,3]
输出: 10

来源：力扣（LeetCode）
链接：https://leetcode-cn.com/problems/largest-rectangle-in-histogram

我的想法

看图中可以发现，一个柱子能够形成的面积是由它两边比它小的柱子决定的。我们最关键的就是找到两边比它小的柱子，从而计算出阴影的宽。

代码实现

暴力求解法（时间超限）

每遍历一个柱子的时候，都要向两边寻找第一个比它小的柱子，并由此计算出面积。由每一个柱子能够形成的最大面积得出面积最大值。

class Solution {
   public int largestRectangleArea(int[] heights) {
        int res = 0;
        for (int i = 0; i < heights.length; i++) {
            int len = roundCenter(i, heights);
            if (heights[i] != 0) {
                res = Math.max(len * heights[i], res);
            }
        }
        return res;
    }

  
    private int roundCenter(int i, int[] arr) {
        int res = 0;
        for (int j = i - 1; j >= 0; j--) {
            if (arr[j] >= arr[i]) {
                res++;
                continue;
            }
            break;
        }
        for (int j = i + 1; j < arr.length; j++) {
            if (arr[j] >= arr[i]) {
                res++;
                continue;
            }
            break;
        }
        return res + 1;
    }
}

空间换时间

我们需要维护两个数组left，right，数组规定如下：

  • left,right 长度于heights数组长度相等

  • left[i] 存放 heights数组中第i个元素左边第一个比它的下标 （ i== 0 时存放 -1）

  • right[i] 存放 heights数组中第i个元素右边第一个比它小的下标 （ 右边没有元素时，right[i] = heights数组长度）

然后我们遍历heights数组，每一个柱子所能够形成的最大面积=(right[i] - left[i] - 1) * heights[i]，求出所有柱子中最大的面积

class Solution {

    public int largestRectangleArea(int[] heights) {
        int len = heights.length;
        // 维护左边第一个小的元素下标
        int[] left = new int[len];
        // 辅助栈
        Stack<Integer> leftStack = new Stack<Integer>();
        for (int i = 0, j = len - 1; i < len; ++i) {
            // 获取左边第一个小的索引
            while (!leftStack.isEmpty() && heights[leftStack.peek()] >= heights[i]) {
                leftStack.pop();
            }
            left[i] = (leftStack.isEmpty() ? -1 : leftStack.peek());
            leftStack.push(i);
        }
        
        leftStack.clear();
        // 维护右边第一个小的元素下标
        int[] right = new int[len];
        for (int i = len - 1; i >= 0; i--) {
            // 获取右边第一个小的索引
            while (!leftStack.isEmpty() && heights[leftStack.peek()] >= heights[i]) {
                leftStack.pop();
            }
            right[i] = (leftStack.isEmpty() ? len : leftStack.peek());
            leftStack.push(i);
        }

        int ans = 0;
        for (int i = 0; i < len; ++i) {
            ans = Math.max(ans, (right[i] - left[i] - 1) * heights[i]);
        }
        return ans;
    }

    public static void main(String[] args) {
        Solution solution = new Solution();
        System.out.println(solution.largestRectangleArea(new int[]{0, 9}));
    }
}
