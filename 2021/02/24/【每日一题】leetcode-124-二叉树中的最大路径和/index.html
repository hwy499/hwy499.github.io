leetcode-124-二叉树中的最大路径和

题目描述

路径被定义为一条从树中任意节点出发，沿父节点-子节点连接，达到任意节点的序列。同一个节点在一条路径序列中至多出现一次 。该路径 至少包含一个 节点，且不一定经过根节点。

路径和 是路径中各节点值的总和。

给你一个二叉树的根节点 root ，返回其 最大路径和 。

示例 1：

输入：root = [1,2,3]
输出：6
解释：最优路径是 2 -> 1 -> 3 ，路径和为 2 + 1 + 3 = 6

示例 2：

输入：root = [-10,9,20,null,null,15,7]
输出：42
解释：最优路径是 15 -> 20 -> 7 ，路径和为 15 + 20 + 7 = 42

提示：

  • 树中节点数目范围是 [1, 3 * 104]
  • -1000 <= Node.val <= 1000

来源：力扣（LeetCode）
链接：https://leetcode-cn.com/problems/binary-tree-maximum-path-sum

我的想法

后续遍历

访问左子树，得到左子树得最大路径leftSubTreeMaxSum，然后在访问右子树得到右子树得最大路径rightSubTreeMaxSum，然后计算此时子树得最大路径和subTreeMaxSum=leftSubTreeMaxSum+rightSubTreeMaxSum+root.val,然后在leftSubTreeMaxSum和rightSubTreeMaxSum选出一条最长的路径，作为此刻子树的最大路径。

代码实现

public class Solution {

    /**
     * 保存最大路径和的答案
     */
    private int ans = Integer.MIN_VALUE;

    public int maxPathSum(TreeNode root) {
        maximumPath(root);
        return ans;
    }

    /**
     * 给定一个根节点，计算出根节点这棵树的最大路径
     *
     * @param root
     * @return
     */
    int maximumPath(TreeNode root) {
        if (root == null) {
            return 0;
        }
        // 获取左子树的最大路径
        int left = Math.max(maximumPath(root.left), 0);
        // 获取右子树的最大路径
        int right = Math.max(maximumPath(root.right), 0);
        // 一个二叉树的最大路径和（没有岔路）
        // 就是根节点到左子树的最大路径+根节点到右子树的最大路径 即（right + left + root.val）
        ans = Math.max(right + left + root.val, ans);
        // 选举出子树最大的路径与根节点的值相加，作为root这棵树的最大路径
        return Math.max(right, left) + root.val;
    }
}
