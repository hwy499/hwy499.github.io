leetcode-面试题17.07.婴儿名字

题目描述

每年，政府都会公布一万个最常见的婴儿名字和它们出现的频率，也就是同名婴儿的数量。有些名字有多种拼法，例如，John 和 Jon 本质上是相同的名字，但被当成了两个名字公布出来。给定两个列表，一个是名字及对应的频率，另一个是本质相同的名字对。设计一个算法打印出每个真实名字的实际频率。注意，如果 John 和 Jon 是相同的，并且 Jon 和 Johnny 相同，则 John 与 Johnny 也相同，即它们有传递和对称性。

在结果列表中，选择 字典序最小 的名字作为真实名字。

示例：

输入：names = [“John(15)”,”Jon(12)”,”Chris(13)”,”Kris(4)”,”Christopher(19)”], synonyms = [“(Jon,John)”,”(John,Johnny)”,”(Chris,Kris)”,”(Chris,Christopher)”]
输出：[“John(27)”,”Chris(36)”]

提示：

  • names.length <= 100000

我的想法

题目中提到了传递性，这时我们就可以考虑用并查集来做这道题，当然标准的并查集得做出改动。

  • 底层数据结构：由于在第二个列表中会出现第一个列表中不存在的名字，我们需要动态维护节点列表，因为节点类型属于字符串，这里我们采用HashMap来存放节点与节点之间的关系，并且再用一个 HashMap 来维护每一个名字的出现频率。
  • 合并：由于选择字典序最小的名字作为真实名字，标准的并查集中按照高度进行合并就要改成按照字典序来进行合并了，两个不同名字合并时也需要进行频率合并。

最终遍历并查集所有的根节点，然后获取对应的出现频率，拼接成符合条件的结果输出。

我的代码

public class Solution {

    // 所有节点集合
    Map<String, String> nodes;
    // 每个节点对应的频率
    Map<String, Integer> frequency;

    public Solution() {
        this.nodes = new HashMap<>();
        this.frequency = new HashMap<>();
    }

    //找到x的根节点
    public String find(String x) {
        if (!nodes.get(x).equals(x)) {
            //路径压缩
            nodes.put(x, find(nodes.get(x)));
        }
        return nodes.get(x);
    }

    public void union(String x, String y) {
        String str1 = find(x), str2 = find(y);
        if (str1.equals(str2))
            return;
        // 字典序小的作为根
        if (str1.compareTo(str2) > 0) {
            nodes.put(str1, str2);
            //人数累加到根节点
            frequency.put(str2, frequency.get(str1) + frequency.get(str2));
        } else {
            nodes.put(str2, str1);
            frequency.put(str1, frequency.get(str2) + frequency.get(str1));
        }
    }

    public String[] trulyMostPopular(String[] names, String[] synonyms) {
        Solution solution = new Solution();

        for (String str : names) {
            int index1 = str.indexOf('('), index2 = str.indexOf(')');
            // 孩子姓名
            String name = str.substring(0, index1);
            // 频率
            int frequency = Integer.valueOf(str.substring(index1 + 1, index2));
            //将节点一一加入到并查集中
            solution.nodes.put(name, name);
            // 把每个名字的频率一一映射
            solution.frequency.put(name, frequency);
        }

        for (String synonym : synonyms) {
            int index = synonym.indexOf(',');
            String name1 = synonym.substring(1, index);
            String name2 = synonym.substring(index + 1, synonym.length() - 1);
            // 对于不存在的孩子姓名做一下处理
            if (!solution.nodes.containsKey(name1)) {
                solution.nodes.put(name1, name1);
                //频率为0
                solution.frequency.put(name1, 0);
            }
            if (!solution.nodes.containsKey(name2)) {
                solution.nodes.put(name2, name2);
                //频率为0
                solution.frequency.put(name2, 0);
            }
            solution.union(name1, name2);
        }

        List<String> res = new ArrayList<>();
        for (String str : names) {
            int index1 = str.indexOf('(');
            String name = str.substring(0, index1);
            // 是否为根节点
            if (name.equals(solution.find(name))) {
                res.add(name + "(" + solution.frequency.get(name) + ")");
            }
        }
        return res.toArray(new String[res.size()]);
    }
}
